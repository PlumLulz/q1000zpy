# q1000zpy
Q1000Z Keygen

Generates a list of possible passwords for the Q1000Z\
ESSID is myqwstXXXX\
Common macs for these modems start with:\
40:4A:03\
C8:6C:87\
CC:5D:4E

python3 q1000z.py cc5d4ef33338
